# Core Beliefs

## Repository Is the System of Record
Repository-local, versioned artifacts are all the agent can see.

## CLAUDE.md Is a Map, Not a Manual
CLAUDE.md stays under 120 lines. It tells agents WHERE to look.

## Plans Are Living Documents
Execution plans accumulate progress, surprises, and drift during implementation.

## Reflect Early and Often
Lightweight reflection after each task catches stale docs immediately.
